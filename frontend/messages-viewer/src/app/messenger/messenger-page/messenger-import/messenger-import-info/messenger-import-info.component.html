1. Login to your Facebook account.
  2. Go to Settings & Privacy > Settings > Your Facebook information > Download your data.
  3. Select range, format JSON and media quality. In your information section select Messages.
  4. Press Create file button.
  5. Wait until file is generated (may took a long time).
  6. Download file.
  7. Go to this page.
  8. Type your Facebook/Messenger name (not pseudonym).
  9. Add previously created and downloaded file (Choose file).
  10. Press start import button.
